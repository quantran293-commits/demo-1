1. Git
    Undo action:
        Undo commit mess: git commit --amend -m”<message mới>” thay đổi commit message mới nhất
        Unstage: git restore --staged <file> đưa 1 file ở staging area về working directory
        un-commit: git reset --soft HEAD~1 đưa nội dung commit cuối về staging
    Branching:
        Tạo nhánh để làm việc rồi push lên chờ review trước khi merge vào master
    Ignore:giúp bạn chỉ định những file và thư mục nào sẽ không được theo dõi (untracked) bởi Git.
